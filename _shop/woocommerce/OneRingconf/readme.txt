Lokale WordPress-/WooCommerce-Integration fuer den ASF 3D-Ringkonfigurator.

== Description ==

Das Plugin stellt den aktuellen WooCommerce-Build des Angular-Konfigurators per Shortcode bereit und verbindet gespeicherte Ring-IDs mit WooCommerce-Warenkorb und Bestellungen.

== Shortcodes ==

* [asf_ringkonfigurator]
* [3D-Trauringkonfigurator]

Optional kann eine vorhandene Konfigurations-ID mit dem Attribut id uebergeben werden.

== Installation ==

Das installierbare ZIP wird mit npm run build:woocommerce erzeugt. Der enthaltene Pluginordner darf in WordPress beliebig umbenannt werden.
